Features

How to Use

To start using our CSS library, simply include the CSS file in your HTML file or link the CSS file directly from our CDN:

            
              <link rel="stylesheet" type="text/css" href="https://cdn.mycsslibrary.com/css/library.css">

You can then start using the various styles and classes in your HTML elements to add the desired effects and styles to your website.

Dark Mode:

Our library also includes a built-in dark mode feature, allowing you to switch between light and dark modes with a single click. The dark mode feature uses a dark, sleek color palette that is perfect for low-light environments and enhances readability. To enable dark mode, simply add the following code to your HTML file:

      
        <button id="dark-mode-toggle">Toggle Dark Mode</button>
        <script>
          const toggleButton = document.getElementById("dark-mode-toggle");
          toggleButton.addEventListener("click", function() {
            document.body.classList.toggle("dark-mode");
          });
        </script>
      </code>
    </pre>
    
In your CSS file, you can define the styles for both light and dark modes:

    
      
        /* Light Mode */
        body {
          background-color: #ffffff;
          color: #000000;
        }

        /* Dark Mode */
        body.dark-mode {
          background-color: #000000;
          color: #ffffff;
        }
      </code>


      
You can customize the color palette to match your website's brand by modifying the CSS code in the example above.
